Options for all browser instances (via account sync)
Proxy host

Twitter API doesn't return CORS headers in the responses, which makes it unusable for the JavaScript code running in a browser. As a workaround, this extension requires you to specify a hostname of a proxy that will forward all requests to api.twitter.com and add CORS headers to the responses.

WARNING: make sure that you trust the proxy, because it will see all your traffic, including access tokens with the same permissions to your Twitter account as this extension.
